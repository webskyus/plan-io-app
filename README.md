# Instruction

| Command      | Description              |
| ------------ | ------------------------ |
| pnpm install | install all dependencies |
| pnpm dev     | run project              |
| pnpm lint    | check with eslint        |
| pnpm build   | build app                |
| pnpm start   | run built app            |
